# DiamondTrainer

Open `index.html` in a modern browser to run the app locally. No build step required.

Files:
- `index.html` — main single-page app
- `style.css` — styling
- `script.js` — main logic
- `utils.js`, `visuals.js`, `data.js` — helpers
- `assets/` — icons and datasets

Quick start:
1. Open `index.html` in Chrome or Firefox.
2. Use the left controls to set inputs and weights.
3. Click **Run Forward Pass** to animate and render LaTeX.
4. Use **Compute Grad** and **Update Weights** to run a single gradient step.

Keyboard shortcuts: Space = Run Forward, → = Step, R = Randomize weights.
